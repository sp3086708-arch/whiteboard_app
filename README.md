# Whiteboard Web Application

An interactive browser-based whiteboard application developed using HTML, CSS, and JavaScript. The application allows users to draw freely on a digital canvas, customize brush colors and sizes, erase content, and save their work as an image. Built using the HTML5 Canvas API, the project provides a smooth and responsive drawing experience with a simple and user-friendly interface.

## Features

* Freehand drawing on a digital canvas
* Multiple color options
* Adjustable brush size
* Eraser tool
* Clear canvas functionality
* Save drawings as images
* Responsive and intuitive user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* HTML5 Canvas API

## Project Objective

The objective of this project is to provide a simple and interactive platform for digital sketching, note-taking, and visual brainstorming while demonstrating frontend web development and Canvas API concepts.
